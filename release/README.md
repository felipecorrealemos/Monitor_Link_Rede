# Pasta de Distribuicao (APK)

Esta pasta e destinada a armazenar manualmente o APK final pronto para instalacao no Android.

Recomendacoes:
- Coloque aqui apenas arquivos finais de distribuicao (ex.: `Monitor_Link_Rede-v1.0.apk`).
- Nao use `bin/` e `obj/` para distribuir o app.
- No futuro, considere publicar instaladores em GitHub Releases.
